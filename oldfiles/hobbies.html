Linux

I first got into contact with Linux around September 2020. I was fed up with Windows for a moment, so I was looking for alternatives... stumbled upon Linux.

My first ever distro that I used was Ubuntu 20.04. I had a little dual boot set up with Windows and Linux for a while, until I realized I found myself more often on Windows because of videogames... I uninstalled Ubuntu.

It doesn't end there. I still experimented a lot in virtual machines.

Then we arrive at the November 2022 incident. Windows updates: You know 'em, right? Yeah, so uh... one of them kind of destroyed my Windows installation. I had a ton of issues before so I just decided "nah, I'll do a full switch NOW" (I already planned to possibly switch when I was getting a new PC)

Distrohopped a bit through debian and Fedora until finding rest at Kubuntu. I love Linux now, one of my favourite things include how it takes up less storage space than Windows and the customizabilty. I can imagine switching to another distro in the future (I'd love to try out something that is a rolling release) but for a while I have been sticking to Kubuntu.

Later in June 2022 I switched to debian. I've been trying out the MATE desktop environment (I did not like it that much) and afterwards the xfce desktop environment (which I fairly like, but I prefer KDE), followed by the GNOME Shell (which is awesome.)

I then switched to Arch with KDE. The installation was painful: Had to install it three times because I destroyed the first two installations both when I was trying to install N(o)VIDE(o)A drivers. But hey. I USE ARCH BTW.
